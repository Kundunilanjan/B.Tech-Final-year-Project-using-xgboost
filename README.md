# Soil Safety Prediction using XGBoost

Machine Learning based soil risk prediction system for safe construction.

## Technologies Used
- Python
- XGBoost
- Streamlit
- Scikit-learn
- Plotly

## Features
- Real-time prediction
- Interactive dashboard
- Probability visualization
- Risk classification
- Streamlit deployment

## Run Locally

pip install -r requirements.txt
streamlit run app.py
